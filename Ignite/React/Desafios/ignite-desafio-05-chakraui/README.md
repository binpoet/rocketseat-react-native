# Desafio Ignite React - 04

Desafio do curso Ignite da Rocketseat.

O desafio era criar um site de viagens.

## Test
Para compilar a aplicação utilize a sequência: 
```
yarn
yarn dev
```

Para testar a aplicação utilize:
```
yarn test
```
